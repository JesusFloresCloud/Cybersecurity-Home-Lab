# Cybersecurity-Home-Lab
Documentation and configurations from my cybersecurity home lab using AWS, Splunk, Active Directory, Sysmon, and Windows Server.

## Overview

This repository documents my cybersecurity home lab designed to practice security monitoring, system administration, and cloud security concepts.

## Lab Environment

### Infrastructure
- AWS EC2
- Windows Server
- Ubuntu Server

### Security Tools
- Splunk Enterprise
- Splunk Universal Forwarder
- Sysmon

### Systems
- Active Directory Domain Controller
- Windows endpoints
- Linux server

## Projects

### Splunk SIEM Implementation

Built a SIEM environment to collect and analyze Windows security telemetry.

Implemented:
- Windows Event Log collection
- Sysmon monitoring
- Authentication monitoring
- Network connection monitoring

### Active Directory Lab

Configured:
- Domain Controller
- Organizational Units
- Users and Groups
- Group Policy Objects

### AWS Cloud Lab

Configured:
- EC2 instances
- IAM roles
- Security groups
- Systems Manager access
